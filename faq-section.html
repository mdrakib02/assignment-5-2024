What is the Document Object Model (DOM)?

Dom is like a map/ structure of everything on a webpage (like text, images, buttons, etc)

How do you select on element from the DOM?

There are some methods to select element. as like 1. getElementById() 2.getElementsByClassName() 3.GetElementsByTagNAme() 4.querySelector() 5.querySelectorAll()

What is event delegation in the context of the dom, and why is it use ful?

Event delegation is a technique in JAVA Script that allows you to handle events ata parent level instead of attaching event listeners to individual child elements. It is use full because , if you have hundreds of item or elements, using event delegation reduce the number of event listeners improving performance.

How do you manipulate and elements attribute and styles using the dom?

We can use various methods. .setAttribute (set an new value for an attribute), .getAttribute (gets the current value of an attribute), .removeAttribute(Removes an attribute from an elements)
